Quiz 03 Practice

Questions

Below are optional practice problems for Quiz 03. These problems are intended to help you become more comfortable working with dictionaries and lists. This practice is intended to reinforce important concepts and does not necessarily reflect the format of the questions on the quiz. Solutions for each problem can be found at the bottom of this page.

Dictionaries T/F

  1. To access a value in a dictionary, you use the format <dictionary_name>{key}. (T/F)
  2. One value can be associated with multiple keys. (T/F)
  3. Dictionaries are mutable. In other words, the values in a dictionary can be changed after they are constructed. (T/F)
  4. Dictionaries can have lists or other dictionaries as their keys. (T/F)
  5. The not in operator can be used to check whether a specific key exists in a dictionary or not. (T/F)
  6. For the purposes of this course, all values in a dictionary are of the same type. (T/F)
